# Zarixsol Ai Dev Studio

A modern AI development studio for building, testing, and deploying AI-powered applications with a desktop and web UI. Built with TypeScript, React, Electron, and a modular architecture.

## Features
- Desktop and web-based UI
- Modular agent and core packages
- Integrated chat, editor, and terminal components
- Project and workspace management
- Secure authentication and permissions

## Getting Started
See [INSTALL.md](INSTALL.md) for setup instructions.

## Project Structure
- `apps/desktop` — Electron main process and UI
- `apps/ui` — Web UI (React + Vite)
- `packages/` — Core, agents, branding, engine, security, services
- `config/` — Configuration files
- `workspace/` — User workspace data
