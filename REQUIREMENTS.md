# Requirements

## System Requirements
- Node.js v18 or above
- pnpm (preferred) or npm/yarn
- Git

## Project Dependencies

### Root
- See `package.json` for root dependencies

### Apps
- `apps/desktop`: Electron, TypeScript
- `apps/ui`: React, Vite, TypeScript

### Packages
- `core`, `agents`, `branding`, `engine`, `security`, `services`: TypeScript, see individual `package.json`

## Installation
Run `pnpm install` at the root to install all dependencies.

## Notes
- For any missing or failed dependencies, see [ISSUES.md](ISSUES.md).
- For development, see [INSTALL.md](INSTALL.md).
